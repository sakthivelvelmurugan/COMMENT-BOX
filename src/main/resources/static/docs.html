Introduction

CommentBox is an AI-powered code annotation tool designed to enrich your source files with developer-friendly explanations. Paste raw logic from Java, Python, or C++, choose your style and density, and let AI generate accurate annotations instantly.

Comment Styles

You can customize how comments are written using the following formatting modes:

  • Inline: Generates concise comments (such as // in Java/C++ or # in Python) positioned directly above or next to key statements.
  • Block: Creates structured paragraph summaries positioned above logical code segments, function definitions, or complex statements.
  • Javadoc / JSDoc: Generates documentation headers (such as /** ... */) for classes and function definitions, combined with concise inline comments inside method bodies.

Density Controls

Configure the verbosity of generated comments based on your target audience:

  • Normal: Annotates meaningful logical blocks and complex operations.
  • Verbose: Explains almost every statement, making it perfect for educational material or onboarding guides.
  • Minimal: Annotates only complex mathematical/logical expressions or non-obvious workarounds.

Usage Limits & BYOK

By default, CommentBox offers 10 free generates per day. If you reach this limit, you can store your own OpenRouter API Key in the secure lock field at the bottom of the editor page. Storing your own key switches the app to BYOK (Bring Your Own Key) mode, unlocking unlimited runs directly through your account.
